# Zepto SQL Data Analysis

## Overview
This project analyzes Zepto's product inventory dataset using PostgreSQL. The goal is to perform data cleaning, exploratory data analysis (EDA), and answer business-related questions using SQL.

## Dataset
The dataset contains information about products available on Zepto, including:
- Category
- Product Name
- MRP
- Discount Percentage
- Discounted Selling Price
- Available Quantity
- Weight (grams)
- Stock Availability

## Objectives
- Import CSV data into PostgreSQL
- Perform data cleaning and validation
- Analyze product inventory
- Answer business questions using SQL
- Practice SQL queries commonly asked in Data Analyst interviews
